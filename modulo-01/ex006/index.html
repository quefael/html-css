Exemplos de semântica

A tag <strong> significa que o termo delimitado possui força dentro da frase. Logo, ele aparecerá em negrito.

A tag <em> significa que queremos dar ênfase (do Inglês emphasis) ao termo. Logo, ele aparecerá em itálico.

A tag <mark> serve para delimitar o texto que queremos demarcar, como se estivéssemos usando uma caneta marcador (podemos alterar os parâmetros para personalizar).

A tag <del> significa que o texto está ali, pode até ser lido, mas deve ser desconsiderado pelo leitor.

A tag <ins> significa que o texto está ali, deve ser lido e você deve prestar atenção nele.

A tag <sup> serve para sobrescrever um texto, como em “x2”.

A tag <sub> serve pra subscrever um texto, como em “H2O”.

Para delimitar trechos de código, utilizamos a tag <code>

Para criar citações utilizamos a tag <q> (quote)

ou <blockquote> (podemos utilizar o parâmetro cite, para linkar algo).

Para escrever uma sigla, mas deixar claro ao usuário (e aos mecanismos de busca) o significado dela, use a tag <abbr>.
